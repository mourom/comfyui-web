# ComfyUI Web

ComfyUI官方网站，基于GitHub Pages和百度BCC服务器构建的前后端分离架构。

## 技术栈
- **前端**：HTML5 + CSS3 + JavaScript
- **后端**：Node.js + Express
- **部署**：GitHub Pages (前端) + 百度BCC (后端)

## 访问地址
- 前端：https://mourom.github.io/comfyui-web
- 后端API：http://120.48.169.183:3000/api

## 功能特性
- 响应式设计，支持各种设备
- 现代化的UI界面，包含彩虹玻璃效果
- 前后端分离架构，API驱动
- 支持HTTPS和HTTP协议
